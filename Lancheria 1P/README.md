# Micro Projeto Lancheria 1P (1 Pastel)

## Instruções para rodar o projeto

Baixe o projeto e execute o seguinte comando no terminal:

```shell
npm install
```

> [!TIP]
> Esse comando irá gerar a pasta `node_modules` com todas as dependências contidas no arquivo  `package.json`

Após a instalação da pasta `node_modules` execute o comando

```shell
npm run dev
```

> [!TIP]
> Esse comando executará o programa através do arquivo `src/main.ts`
